Find the k-th number that's missing

Given a sorted array of distinct positive integers and a number k, return the k-th positive integer that is not in the array.

The key insight → For a strictly increasing array, the value at index i would be i+1 if nothing were missing (1,2,3,...). So the count of missing numbers at or before arr[i] is exactly arr[i] − (i+1). That single formula turns this into a clean binary search: find the first index where the missing-count reaches k.
